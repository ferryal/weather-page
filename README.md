## Weather Web App

## Tech Stack

1. React
1. Redux 
1. Axios
1. NodeJs
1. NPM
1. Yarn


## How to run this project

### Install Package

```
npm install / yarn install
```

### Setup Development Mode

```
npm start / yarn start
```

```
The local server will live at port 3000 by default
```

## How to build this project

```
npm build / yarn build
```
